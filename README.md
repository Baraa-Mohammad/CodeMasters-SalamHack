# إثمار - Ithmar

إثمار منصة تمويل جماعي ذكية لدعم النساء وأصحاب المشاريع الصغيرة في فلسطين. تربط المنصة أصحاب المشاريع مع الشركات والمؤسسات الداعمة، ثم تضمن استخدام التمويل بشكل صحيح من خلال محفظة رقمية، دفع تجريبي عبر QR، ولوحة تتبع شفافة للمصاريف.

## المشكلة

تواجه النساء وأصحاب المشاريع الصغيرة في فلسطين صعوبة في الوصول إلى التمويل بسبب ضعف الثقة بين الممول وصاحب المشروع، وغياب الشفافية بعد وصول المال، وسوء استخدام التمويل في بعض الحالات، مما يؤدي إلى تعثر مشاريع واعدة.

## الحل

إثمار تجعل مسار التمويل واضحاً:

1. صاحب المشروع ينشئ حملة تمويل.
2. الإدارة تراجع المشروع وتوافق عليه.
3. الممول يدعم المشروع من السوق.
4. التمويل يصل إلى محفظة صاحب المشروع.
5. صاحب المشروع يصرف المال عبر QR لموردين.
6. كل عملية صرف تظهر في تقرير شفافية.

## المميزات

- واجهة عربية RTL كاملة.
- Landing Page احترافية مع شعار إثمار.
- داشبورد مخصص لكل دور: صاحب مشروع، ممول، مورد، إدارة.
- سوق مشاريع مع بحث وفلترة حسب التصنيف وRisk Score.
- إنشاء مشاريع وحفظها في MySQL.
- مراجعة إدارية للمشاريع قبل ظهورها في السوق.
- تمويل مشروع وتحديث نسبة التمويل والمحفظة.
- دفع QR تجريبي يخصم من المحفظة.
- Tracking Dashboard يوضح أين صُرف المال.
- رسائل نجاح وخطأ وحالات فارغة مناسبة للعرض.

## الأدوار

- `project_owner`: إنشاء مشاريع، متابعة التمويل، إدارة المحفظة، الدفع عبر QR.
- `funder`: استعراض المشاريع، تمويل المشاريع، متابعة تقارير الشفافية.
- `supplier`: استقبال مدفوعات QR ومشاهدة سجل العمليات.
- `admin`: مراجعة المشاريع، الموافقة والرفض، مراقبة المستخدمين والنشاط.

## التشغيل على XAMPP

1. ضع مجلد المشروع هنا:

```text
C:\xampp\htdocs\code\ithmar
```

2. شغل Apache وMySQL من XAMPP Control Panel.

3. افتح phpMyAdmin:

```text
http://localhost/phpmyadmin
```

4. اختر Import واستورد الملف:

```text
C:\xampp\htdocs\code\ithmar\database.sql
```

5. افتح الموقع:

```text
http://localhost/code/ithmar/
```

## Golden Flow للعرض النهائي

1. سجل الدخول كصاحب مشروع: `owner@ithmar.ps`.
2. أنشئ مشروعاً جديداً. سيظهر بحالة `قيد المراجعة`.
3. سجل الدخول كمدير: `admin@ithmar.ps`.
4. افتح لوحة الإدارة ووافق على المشروع.
5. سجل الدخول كممول: `funder@ithmar.ps`.
6. افتح سوق المشاريع وموّل المشروع.
7. عد إلى صاحب المشروع وشاهد زيادة رصيد المحفظة.
8. ادفع لمورد من صفحة الدفع عبر QR.
9. افتح Tracking Dashboard وشاهد العملية ضمن تقرير الشفافية.

## صفحات العرض

- الصفحة الرئيسية: تشرح الفكرة، المشكلة، وآلية العمل.
- سوق المشاريع: يعرض مشاريع فلسطينية صغيرة ببطاقات وتمويل وتصفية.
- لوحة صاحب المشروع: تركز على التمويل والمحفظة والصرف.
- لوحة الممول: تعرض المشاريع المقترحة والتمويلات السابقة.
- لوحة المورد: تعرض QR وسجل المدفوعات.
- لوحة الإدارة: تعرض الإحصائيات ومراجعة المشاريع والمستخدمين.
