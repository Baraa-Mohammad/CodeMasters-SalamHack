# 📋 توثيق الميزات الجديدة - منصة إثمار

## الصفحات الجديدة المضافة

### 1. **صفحة الملف الشخصي** (`pages/profile.php`)
- **الوصول**: جميع المستخدمين المسجلين
- **الميزات**:
  - عرض بيانات الحساب الشخصي
  - تحديث الاسم والهاتف
  - عرض إحصائيات حسب نوع الحساب:
    - **صاحب المشروع**: عدد المشاريع، إجمالي التمويل، رصيد المحفظة
    - **الممول**: عدد المشاريع الممولة، إجمالي الدعم المالي
    - **المورد**: عدد المعاملات، إجمالي الدخل
  - إعدادات الإشعارات
  - منطقة الخطر (تسجيل الخروج)

### 2. **لوحة الممول المحسّنة** (`pages/dashboard-funder.php`)
- **الوصول**: الممولون والمؤسسات فقط
- **التحسينات**:
  - عرض المشاريع الممولة بشكل تفصيلي
  - سجل شامل للمعاملات
  - توزيع التمويل حسب الفئات
  - مشاريع مقترحة بناءً على المخاطر
  - مؤشرات التأثير الاجتماعي
  - نصائح التمويل المخصصة

### 3. **صفحة تفاصيل المورد** (`pages/supplier-details.php`)
- **الوصول**: جميع المستخدمين
- **الميزات**:
  - معلومات المورد الكاملة
  - إحصائيات الأداء (المعاملات، الدخل، المشاريع)
  - سجل المعاملات التفصيلي
  - قائمة المشاريع المدعومة
  - توزيع الفئات المخدومة
  - حالة التوثيق والموثوقية

### 4. **صفحة الإشعارات** (`pages/notifications.php`)
- **الوصول**: جميع المستخدمين المسجلين
- **الإشعارات حسب الدور**:
  - **صاحب المشروع**: حالة المشاريع، التمويلات، مدفوعات QR
  - **الممول**: مشاريع جديدة، تحديثات التمويل
  - **المورد**: معاملات دفع جديدة
  - **المسؤول**: مشاريع قيد المراجعة، إحصائيات النظام
- **الميزات**:
  - تصفية الإشعارات
  - عرض الوقت النسبي (الآن، دقائق، ساعات)
  - روابط مباشرة للتفاصيل
  - تصميم قابل للتفاعل

### 5. **صفحة التقارير والإحصائيات** (`pages/reports.php`)
- **الوصول**: المسؤولون فقط
- **الإحصائيات الشاملة**:
  - إجمالي المستخدمين والمشاريع والتمويلات
  - توزيع حالة المشاريع (قيد المراجعة، مقبول، ممول، مرفوض)
  - توزيع المستخدمين حسب النوع
  - تقييم المخاطر (منخفض، متوسط، مرتفع)
  - الفئات الأكثر تمويلاً
  - أكثر 10 مشاريع نشاطاً
  - سجل الأنشطة الأخيرة

## التحسينات على الوظائف المساعدة

### الدوال المضافة في `includes/helpers.php`:

```php
ensure_wallet($userId)          // التأكد من وجود محفظة للمستخدم
can_view_project($user, $proj)  // التحقق من صلاحية عرض المشروع
get_user_projects_count()       // الحصول على عدد مشاريع المستخدم
get_funding_status($project)    // الحصول على حالة التمويل
category_icon($category)         // الحصول على أيقونة الفئة
```

## تحسينات الأمان والصلاحيات

### صلاحيات الوصول:
- ✅ صاحب المشروع: يرى فقط مشاريعه والمشاريع المقبولة
- ✅ الممول: يرى المشاريع المقبولة فقط
- ✅ المورد: يرى المعاملات الخاصة به
- ✅ المسؤول: يرى جميع البيانات

### الحماية:
- استخدام `require_login()` للتحقق من تسجيل الدخول
- استخدام `require_role()` للتحقق من الصلاحيات
- تنظيف الإدخالات باستخدام `e()` (htmlspecialchars)

## التدفق المتقدم للمستخدمين

### صاحب المشروع:
1. **الملف الشخصي** → مراجعة البيانات
2. **لوحة التحكم** → عرض المشاريع والتمويلات
3. **الإشعارات** → تتبع المستجدات
4. **المحفظة** → إدارة الأموال
5. **الدفع عبر QR** → صرف الأموال
6. **التتبع** → مراقبة الشفافية

### الممول:
1. **الملف الشخصي** → إدارة الحساب
2. **لوحة التحكم** → عرض الاستثمارات
3. **المتجر** → البحث عن مشاريع جديدة
4. **الإشعارات** → متابعة التطورات
5. **التقارير** (في لوحتي الممول والمسؤول)

### المسؤول:
1. **لوحة التحكم** → مراجعة المشاريع
2. **التقارير** → تحليلات شاملة
3. **الإشعارات** → متابعة النشاطات
4. **الملف الشخصي** → إدارة الحساب
