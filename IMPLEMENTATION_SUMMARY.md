### 2️⃣ صفحة تفاصيل المورد (`supplier-details.php`)

**الميزات**:

**الوصول**: جميع المستخدمين

### 3️⃣ صفحة الإشعارات (`notifications.php`)
**الهدف**: إدارة وتتبع الإشعارات

**الإشعارات حسب الدور**:
- **صاحب المشروع**: حالة المشاريع، التمويلات الجديدة، مدفوعات QR

- **المورد**: معاملات دفع جديدة، تحديثات الأداء
- **المسؤول**: مشاريع قيد المراجعة، إحصائيات النظام

**الميزات**:
- تصفية الإشعارات
- عرض الوقت النسبي (الآن، دقائق، ساعات)
- روابط مباشرة للتفاصيل

**الوصول**: جميع المستخدمين المسجلين

## 🔒 الأمان والصلاحيات

### التحقق من الصلاحيات:
```php
// التحقق من تسجيل الدخول
$user = require_login();

// التحقق من النوع (الدور)
$user = require_role(['project_owner', 'funder']);

// التحقق من الوصول للمشروع
if (!can_view_project($user, $project)) {
    flash('error', 'صلاحيات غير كافية');
    redirect('index.php');
}
```

### تنظيف الإدخالات:
- استخدام `e()` (htmlspecialchars) لجميع المخرجات
- استخدام Prepared Statements لجميع الاستعلامات
- التحقق من الأنواع باستخدام type hints
